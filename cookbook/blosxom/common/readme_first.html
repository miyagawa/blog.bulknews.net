About This Blog (Blog Developer's Cookbook)

RSS, Trackback, Weblogs.com Ping, Blogger API などの Weblog コアテクノロジや、 Google API, Amazon Web Service (AWS) といった関連する Web サービスをネタにして、プログラミングテクニックを Cookbook 形式で紹介していきます。言語は主に Perl を使いますが、Ruby あたりもターゲットとして考えてます。

最終的には書籍化をもくろんではいますがそもそもちゃんと書けるのかどうか。いつもギリギリに書きはじめる怠惰な自分のモチベーションを保つため、Blog 形式で書籍を執筆することに挑戦したいという意味合いもあります。blosxom でコードサンプルをエントリに埋め込むプラグインとか作ったら結構書籍執筆用の Blog ツールとしては便利かもしれません。この辺の柔軟性とかも解説していきたいと思っています。
