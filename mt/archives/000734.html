Bulkfeeds: Search v2.0 のベータ版公開中

Bulkfeeds Search2 ベータ版

Bulkfeeds の全文検索に Google 風の検索オプションをサポートした ver. 2.0 をベータリリースしました。

概要

Bulkfeeds Search ver 2.0 では、以下のようなシンタクスをサポートしています。
XML site:bulknews.net
XML という単語を含む、bulknews.net ドメイン内の記事
"Movable Type" -site:bulknews.net
"Movable Type" というフレーズを含む、bulknews.net ドメイン以外の記事
Bulknews -title:Bulkfeeds rssurl:http://blog.bulknews.net/mt/index.rdf
Bulknews という単語を含み、タイトルに Bulkfeeds という単語を含まない、RSS の URL が http://blog.bulknews.net/mt/index.rdf 内の記事
これによっていままで Bulkfeeds Search では実現できなかった、
  • 空白を含むフレーズ検索 ("foo bar")
  • NOT 検索 (-)
  • OR 検索 (|)
に加え、
  • 検索するドメインの絞込み (- によって NOT も使用可)
  • rssurl 指定による Blog 内検索
のようなことも実現できるようになりました。

検索フィールドとして利用できるオプションは
  • rssurl: RSS Feed の URL
  • site: 記事の permalink の URL
  • title: 記事の title
の 3つです。

特定のBlog サービスからのみの検索 (e.g. site:-nifty.com) や、逆に特定のドメインを結果から除去しての検索 (e.g. livedoor -site:livedoor.jp) などに利用すると便利かとおもいます。

また rssurl, site については search2b へのクエリパラメータとして渡すこともできます (例) ので、hidden パラメータや radio ボタンを使って Bulkfeeds サイト内検索ボックスを設置することも可能になります（後述）。

NOTE

Bulkfeeds Search v2.0 はベータ版としてリリースしますので、以下の点にご注意してご利用ください。ご意見、ご質問などはこのエントリまでおねがいします。
  1. URL (/app/search2b) はリリース時に変更になりますので、サイト内検索の設置などはとりあえずしないでください。
  2. 同様の理由により、検索結果の RSS Feed は現在用意していません。シンタクス等のベータテストやバグレポートにご利用ください。
  3. 現在稼働中の search2 と search2b は、見ている記事データベースが異なります。search2b については、1/21 0:00 ごろからアグリゲートした RSS Feed のみがインデクシングされていますので、search2 で検索できても、search2b ではできない記事が相当数あります。(はやく search2b の検索結果にいれるには、Ping フォーム に RSS Feed の URL をいれてください。) リリース時には、search2 の記事データベースを移行する予定です。
